Kirim notifikasi rekap absensi otomatis ke grup Telegram ustadz. Gratis tanpa batas.
Cara mendapatkan Token & Chat ID
1. Buat Bot:
Buka Telegram → cari @BotFather → ketik /newbot → ikuti instruksi → copy Token

2. Tambahkan bot ke grup:
Tambahkan bot sebagai anggota grup ustadz, lalu jadikan Admin

3. Dapatkan Chat ID:
Kirim pesan apapun di grup → buka browser:
https://api.telegram.org/bot<TOKEN>/getUpdates
Cari angka "id" di dalam "chat" (biasanya diawali -100...)
